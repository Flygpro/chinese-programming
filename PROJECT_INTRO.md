# 项目介绍

## 🐉 中文编程示例项目

> **用中文写代码，让编程更亲切！**

---

## 项目背景

编程语言大多以英文关键字为基础，对中文母语者来说存在一定门槛。本项目旨在通过 **Python 中文编程** 的方式，展示如何用中文变量名、函数名、类名来编写规范、可运行的代码，降低初学者的理解成本，让编程变得更亲切。

---

## 项目特色

- 🀄 **全中文命名** —— 变量、函数、类、方法全部使用中文
- 📖 **注释详尽** —— 每个示例都有完整的中文注释说明
- 🎮 **实战导向** —— 包含计算器、猜数字等可运行的小项目
- 📚 **循序渐进** —— 从基础语法到面向对象，由浅入深
- ✅ **即开即用** —— 纯 Python 标准库，无需额外依赖

## 示例内容一览

### 1. hello_world.py
最基础的入门程序，演示中文变量赋值与 f-string 格式化输出。

### 2. 函数和类.py
- 中文函数定义与调用
- 中文类的创建、继承与多态
- `__init__`、`__str__` 等魔术方法的中文使用

### 3. 数据结构.py
- 列表的增删改查与推导式
- 字典的遍历与推导式
- 集合的交集、并集、差集运算
- 元组的定义与解包

### 4. 异常处理.py
- `try-except-else-finally` 中文版
- 自定义异常类
- 安全的类型转换与除法

### 5. 模块导入.py
- 使用 `import ... as 中文别名` 导入标准库
- 封装实用函数：掷骰子、日期查询、勾股定理

### 6. 实战示例/中文计算器.py
命令行交互式计算器，支持四则运算和括号，包含输入校验与帮助系统。

### 7. 实战示例/猜数字.py
完整的猜数字小游戏，支持难度调节、尝试次数限制、评级系统和循环游玩。

## 快速开始

```bash
# 克隆项目
git clone https://github.com/你的用户名/chinese-programming.git
cd chinese-programming

# 运行基础示例
python3 examples/hello_world.py

# 运行猜数字游戏
python3 examples/实战示例/猜数字.py

# 运行计算器
python3 examples/实战示例/中文计算器.py
```
